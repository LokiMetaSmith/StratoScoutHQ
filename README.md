# StratoScout — Marketing Website

## Project Overview
StoryBrand-structured landing page for StratoScout, a high-altitude UAS geospatial intelligence startup targeting county emergency managers for wildfire prevention.

## Completed Features
- Hero section with background wildfire imagery, tagline, and stat bar
- Problem section with NWS Tulsa quote
- Solution section with AI analysis callout and 4-feature grid
- How It Works step-by-step flow (4 steps)
- Demo section with emergency manager imagery
- Validation strip (3 cards)
- Contact form with success state
- Fixed nav with scroll-aware active states
- Fade-in scroll animations
- Fully responsive (mobile/tablet/desktop)

## Entry Points
- `/index.html` — Main landing page

## StoryBrand Structure
1. **Hero** — Character + Stakes
2. **Problem** — External + Internal + Philosophical
3. **Solution** — Guide with a plan
4. **Demo** — Call to Action
5. **Contact** — Direct CTA: Get in Touch
